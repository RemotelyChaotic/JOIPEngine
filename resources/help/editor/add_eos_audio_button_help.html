Add Audio Instruction

This button adds an audio instruction to the currently opened eos script in the EOS Editor.


The following parameters can be changed for audio instructions:

  • locator (json attr: locator): The button next to the line editor lets you select the Sound Resource to load and play.
  • id (json attr: id): This is the field where an id can be entered.
    This id can be used in eval scripts to access the Sound Resource.
    See the EOS API Reference for more information on how to use sound resources.
  • loops (json attr: loops): Sets the number of times the Sound Resource is replayed when started.
  • start (json attr: startAt): Sets the start frame of the Sound Resource.
  • seek (json attr: seek): Allows you to seek the position of a specific frame in a Sound Resource.
  • volume (json attr: volume): Allows to set the volume for the Sound Resource.


SettingsResource Selection

A full node in serialised JSON form:

{
    "audio.play": {
        "locator": "myBundle/myResource",
        "id": "sound",
        "loops": -1,
        "startAt": 0,
        "seek": 0,
        "volume": 1.0
    }
}
