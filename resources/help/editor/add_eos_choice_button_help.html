Add Metronome Code

This button opens an overlay to add code snippets to the currently opened script in the Script Editor, to interact with the metronome.


For more information about the script API, please see the API Reference.


The overlay has two tabs that each serve to insert different peices of code.



Add Metronome Code Dialog (General Tab)


The first tab contains the general metronome settings like Bpm, beat Pattern and options to stop or start the metronome. If Set BPM (beats / min) is checked, the bpm to the right will be set. If Set pattern is checked, the pattern defined in the table below will be set as pattern. The pattern list can be modified with the "+" and "-" buttons on the side. Clicking "+" will add a new button to the list and clicking "-" will remove the currently selected one.

  • Generated code example for Set BPM (beats / min): metronome.setBpm(90);
  • Generated code example for Set pattern: metronome.setPattern([1,2,2]);
  • Generated code example for Start: metronome.start();
  • Generated code example for Stop: metronome.stop();



Add Metronome Code Dialog (Beat Sound Tab)


The second tab contains a tree to browse Resources including a search bar similar to the Resource Tree in the Resource Manager. Clicking a tree item will select the resource for the generated code snippet. Checking Set resource will generate code to set the currently selected Resource as the beat sound effect for future calls to start. If no Sound type resource is selected, checking the checkbox will generate code to reset the sound to the default one.
To mute or unmute the metronome, the Mute checkbox can be checked or unchecked. Volume controls the volume of the metronome. Please note that the volume is not an absolute value, but multiplied with the engine volume.

  • Generated code example for Set resource: metronome.setBeatResource("thing"); // valid resource selected
  • Generated code example for Set resource: metronome.setBeatResource(null); // invalid or no resource selected
