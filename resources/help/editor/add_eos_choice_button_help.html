Add Choice Instruction

This button adds a choice instruction to the currently opened eos script in the EOS Editor.


The following parameters can be changed for choice instructions:

  • Options Table (json attr: options): The rows of this table describe the buttons, how they look, and their labels. The + Button adds a new button and the - button removes the selected button. The attributes of each row are:
    • Label (json attr: label): The button label. Can be html and can contain evaluator expressions: <eval>//code here</eval>.
    • Color (json attr: color): The button background color in r/g/b.
      Clicking the colored rectangle to the right will open a color picker dialogue.
    • Checkbox (json attr: visible): visibility of the button.


Dialog UI

A full node in serialised JSON form:

{
    "choice": {
        "options": [
            {
                "label": "Option 1",
                "commands": [
                    ...
                ]
                "color": "#ff0000",
                "visible": true
            },
            ...
        ]
    }
}
