Add Disable Instruction

This button adds a disable instruction to the currently opened eos script in the EOS Editor.


The following parameters can be changed for disable instructions:

  • target (json attr: target): The scene you want to disable. Disabled scenes are skipped during next scene resolution after the currently running scene ends and can not be the target of a goto instruction.


Dialog UI

A full node in serialised JSON form:

{
    "disable": {
        "target": "scene_1"
    }
}
