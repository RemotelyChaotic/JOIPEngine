Add Enable Instruction

This button adds an enable instruction to the currently opened eos script in the EOS Editor.


The following parameters can be changed for enable instructions:

  • target (json attr: target): The scene you want to enable. Disabled scenes are skipped during next scene resolution after the currently running scene ends and can not be the target of a goto instruction. This instruction enables previously disabled scenes.


Dialogue UI

A full node in serialised JSON form:

{
    "enable": {
        "target": "scene_1"
    }
}
