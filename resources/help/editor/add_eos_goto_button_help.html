Add Goto Instruction

This button adds a goto instruction to the currently opened eos script in the EOS Editor.


The goto instructions lets you jump to a certain scene. The following parameter can be changed for goto instructions:

  • target (json attr: target): The scene you want to jump to. After a goto instruction the current execution branch ends.


Dialogue UI

A full node in serialised JSON form:

{
    "goto": {
        "target": "scene_1"
    }
}
