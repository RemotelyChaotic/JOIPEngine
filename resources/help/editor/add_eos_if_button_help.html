Add If Instruction

This button adds an if instruction to the currently opened eos script in the EOS Editor.


The if instruction allows to conditionally execute instructions. The following parameters can be changed for enable instructions:

  • target (json attr: target): The entered condition will be executed like an Eval Instruction, except that the return value will be checked. If trouthy, execution will continue with the commands branch; if not the elseCommands branch will be executed.


Dialog UI

A full node in serialised JSON form:

{
    "if": {
        "condition": "runCounter == 1",
        "commands": [
            {
                ...
            }
        ],
        "elseCommands": [
            {
                ....
            }
        ]
    }
}
