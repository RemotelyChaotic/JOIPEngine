Add Image Instruction

This button adds an image instruction to the currently opened eos script in the EOS Editor.


The following parameters can be changed for enable instructions:

  • locator (json attr: locator): The button next to the line editor lets you select the Image or Movie Resource to load or play.


SettingsResource Selection

A full node in serialised JSON form:

{
    "image": {
        "locator": "myBundle/myResource"
    }
}
