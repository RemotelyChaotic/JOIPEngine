Add Prompt Instruction

This button adds a prompt instruction to the currently opened eos script in the EOS Editor.


The following parameters can be changed for choice instructions:

  • variable (json attr: variable): The variable name in which the input will be stored. The variable can be later accessed from Eval Instructions. As such the string needs co be a valid variable name according to the ECMAScript 5 specification.


Dialogue UI

A full node in serialised JSON form:

{
    "prompt": {
        "variable": "name"
    }
}
