Add Timer Instruction

This button adds a timer instruction to the currently opened eos script in the EOS Editor.


The following parameters can be changed for choice instructions:

  • duration (json attr: variable): The duration for which the timer is active. Needs to follow the format <weeks>w<days>d<hours>h<minutes>m<seconds>s<milis>ms or be a plain number denoting miliseconds. For example: 2m30s or 100.
  • isAsync (json attr: align): This flag makes the timer asynchronous. Asynchronous timers are running in a separate thread and the commands will be executed asynchronously when the timer ends. Otherwise the instruction blocks the execution until the timer runs out.
  • style (json attr: align): Determines the stye of the shown timer.
    • normal: Default mode. Remaining time is shown and the timer is visible.
    • hidden: Timer is not visible.
    • secret: Timer is visible but the remaining time is not shown.

A full node in serialised JSON form:

{
  "timer": {
      "duration": "20s",
      "isAsync": false,
