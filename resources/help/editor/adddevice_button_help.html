Add Device Code

This button opens an overlay to add code snippets to the currently opened script in the Script Editor, to control connected devices (if available and connected).


For more information about the script API, please see the API Reference.



Add Vibrate Code Dialog Page


The vibration speed for the generated code can be configured in the spinn box. This page will only generate code if the checkbox is chcked.

  • Generated code example: deviceController.sendVibrateCmd(10.0);



Add Linear Code Dialog Page


The time and length for the generated linear command code can be configured in the spinn box. This page will only generate code if the checkbox is chcked.

  • Generated code example: deviceController.sendLinearCmd(10, 100.0);



Add Roate Code Dialog Page


The rotation speed for the generated code can be configured in the spinn box and the checkbox configures if the rotation is clock-wise or counter-clockwise. This page will only generate code if the checkbox is chcked.

  • Generated code example: deviceController.sendRotateCmd(true, 10.0);



Add Stop Code Dialog Page


This page will generate code for a stop command if the checkbox is chcked.

  • Generated code example: deviceController.sendStopCmd();



Not all devices support all commands.
If a command is not supported by the device, it will have no effect.
