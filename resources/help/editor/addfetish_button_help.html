Fetish Search Bar

In the Fetish Search Bar you can enter a fetish / kink that you want to tag the JOI-Project with. The kinks / fetishes are categorized. You can start searching a fettish by first typing a category and select from the suggested categories. Afterwards enter a '/' to list the fetishes under that category.


Alternatively, clicking the button next to the Search Bar will show the Kink / Fetish selection overlay.



Kink / Fetish selection overlay


The kinks / fetishes and categories can be viewed in the overlay and filtered by entering a regex expression. Any number of kinks / fetishes can also be selected in the overlay.
