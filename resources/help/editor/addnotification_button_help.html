Add Notification Code

This button opens an overlay to add code snippets to the currently opened script in the Script Editor, to show an interactable notification popup.


For more information about the script API, please see the API Reference.



Add Notification Content


The first page lets you Show or Hide notifications or Clear all shown notifications. You can also asign an ID to each notification and optionally set the Timeout and Icon alignment. The Text and Widget Text boxes can be filled, or left empty if no code should be generated for the widget or the text above the widget. The text can contain html for formating (see API Reference for more detail).

  • Generated code example: notification.show("myId","Text",""); // Only text
  • Generated code example: notification.show("myId","","Button"); // Only button
  • Generated code example: notification.show("myId","Text","",20); // Text with timeout of 20s
  • Generated code example: notification.setIconAlignment(IconAlignment.AlignLeft); // set icon alignment



Add Notification Portrait


The second page lets you customize the portrait of the notificatons with a Resource.
The overlay contains a tree to browse Resources including a search bar similar to the Resource Tree in the Resource Manager. Clicking a tree item will select the resource for the generated code snippet. Clicking the x button next to the selected resource will remove it. The selected resource behaves the same as it does for Icons.

  • Generated code example: notification.setPortrait("myResource"); // Resource name

Add Notification on-timeout Script


The third and fourth page behave the same as the second page and let you select resources for the scripts to be called on click of the button or on timeout.

  • Generated code example: notification.show("myId","Text","",20,"timeoutScript.js");
  • Generated code example: notification.show("myId","Text","",20,"timeoutScript.js","clickScript.js");


Add Notification Colors

The last page lets you configure the colors of the notification and the widget. The components can be entered numerically (red, green, blue, alpha in this order). Alternatively, clicking the colored boxes to the right will open a color wheel to pick the color.

  • Generated code example: notification.setTextBackgroundColor([255,0,0,255]); // red text background
  • Generated code example: notification.setTextColor([0,255,0,255]); // green text
  • Generated code example: notification.setWidgetBackgroundColor([0,0,0,0]); // transparent widget/button background
  • Generated code example: notification.setWidgetColor([0,0,255,255]); // blue button text
