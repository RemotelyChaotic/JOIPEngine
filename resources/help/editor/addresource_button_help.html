Add Resource

Open the native file browser to add a Resources to the JOI-Project.


You can select which type of Resources to add on the bottom right.



Resource type selection


If a new Resource is outside the project directory, when it is added to the JOI-Project, a prompt will ask to copy or move the files, I usually copy them to have a backup.



File outside directory prompt


An alternative to adding Resources with this button, is to drag-and-drop the files from the explorer into the Resource Tree.
