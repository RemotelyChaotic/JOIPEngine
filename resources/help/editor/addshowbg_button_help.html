Add Background Code

This button opens an overlay to add code snippets to the currently opened script in the Script Editor, to change the background in the player.


For more information about the script API, please see the API Reference.



Add Background Code Dialog


The overlay contains a tree to browse Resources including a search bar similar to the Resource Tree in the Resource Manager. Clicking a tree item will select the resource for the generated code snippet.


Checking the Resource check box will add code to change the background image to the selected resource.

  • Generated code example: background.setBackgroundTexture("a");


Checking the Color check box will add code to change the background color to the selected color. The color components are in the order red, green, blue and alpha. Clicking the color itself on the right will open a color wheel to freely select the color.

  • Generated code example: background.setBackgroundColor([5,123,0,255]);
