Add Icon Code

This button opens an overlay to add code snippets to the currently opened script in the Script Editor, to show or hide icons in the player.


For more information about the script API, please see the API Reference.



Add Icon Code Dialog


The overlay contains a tree to browse Resources including a search bar similar to the Resource Tree in the Resource Manager. Clicking a tree item will select the resource for the generated code snippet. Clicking the x button next to the selected resource will remove it.


The checkboxes Show and Hide are exclusive and will either add code to show the selected Resource as an icon or hide it. Having no Resource selected and checking Hide will generate code to hide all icons.

  • Generated code example for Show: icon.show("a");
  • Generated code example for Hide: icon.hide("a");
  • Generated code example for Hide: icon.hide("~all"); // in case no Resource is selected
