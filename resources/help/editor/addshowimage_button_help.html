Add MediaPlayer Code

This button opens an overlay to add code snippets to the currently opened script in the Script Editor, to show media files and manipulate them.


For more information about the script API, please see the API Reference.



Add MediaPlayer Code Dialog


The overlay contains a tree to browse Resources including a search bar similar to the Resource Tree in the Resource Manager. Clicking a tree item will select the resource for the generated code snippet.


Checking the Show / Play check box will add code to either show or play the resource. Movie and Sound type Resources will be played, while Image types will be shown. checking Seek will generate code to seek the media file in case of Movie or Sound resources. The input for Seek is in frames. Checking Loops and entering a number > 0 will generate code to loop the media if possible, and Start At and Stop At will generate code to start or stop the media at the given frame.

  • Generated code example for Image: mediaPlayer.show("a");
  • Generated code example for Movie: mediaPlayer.play("b");


Checking the Set volume checkbox allows you to control the volume of the Resource. Please note that the volume is not an absolute value, but multiplied with the engine volume.

Checking the Pause check box will add code to either pause the resource. Only Movie and Sound type Resources will generate any code with this option checked.

  • Generated code example for Movie: mediaPlayer.pauseVideo();
  • Generated code example for Sound: mediaPlayer.pauseSound();


Checking the Stop check box will add code to stop the resource playback. Only Movie and Sound type Resources will generate any code with this option checked.

  • Generated code example for Movie: mediaPlayer.stopVideo();
  • Generated code example for Sound: mediaPlayer.stopSound();


Checking the Wait for finish check box will add code to wait for the currently playing playback.

  • Generated code example: mediaPlayer.waitForPlayback();
