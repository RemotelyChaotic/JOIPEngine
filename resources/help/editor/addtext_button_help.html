Add Text Code

This button opens an overlay to add code snippets to the currently opened script in the Script Editor, to show text or prompt user input.


For more information about the script API, please see the API Reference.


The overlay has four tabs that each serve to insert different peices of code.



Add Text Code Dialog (Text Tab)


The first tab contains a text editor that lets you enter the text to be displayed. Anything entered will generate code to show the text in a single textbox if the Show text check box is checked. If Show user input is checked, code will be generated to prompt for user input instead. If Set Text Alignment is set, code will be generated to set text alignment.

  • Generated code example for Show text: textBox.showText("Hello World!");
  • Generated code example for Show user input: var sInput = textBox.showInput(); // TODO: change variable name
  • Generated code example for Set text alignment: textBox.setTextAlignment(TextAlignment.AlignCenter);



Add Text Code Dialog (Text Tab)


The second tab contains a tree to browse Resources including a search bar similar to the Resource Tree in the Resource Manager. Clicking a tree item will select the resource for the generated code snippet. Checking Set portrait will generate code to set the currently selected Resource as a portrait for future calls to showText. If no Image or Movie type resource is selected, checking the checkbox will generate code to remove the portrait.

  • Generated code example for Set portrait: textBox.setTextPortrait("thing"); // valid resource selected
  • Generated code example for Set portrait: textBox.setTextPortrait(null); // invalid or no resource selected



Add Text Code Dialog (Buttons Tab)


The second tab contains a list that can be modified with the "+" and "-" buttons on the side. Clicking "+" will add a new button to the list and clicking "-" will remove the currently selected one. Double clicking the button name allows for changing the button text. If Show buttons is checked this will generate code for a set of buttons with the configured button texts.

  • Generated code example for Show buttons: var iSelection = textBox.showButtonPrompts(["Yes","No"]); // TODO: change variable name



Add Text Code Dialog (Text Color Tab)


The third tab contains a list that can be modified with the "+" and "-" buttons on the side. Clicking "+" will add a new color to the list and clicking "-" will remove the currently selected one. The color components are in the order red, green and blue. Clicking the color itself on the right will open a color wheel to freely select the color. If Set text colors is checked this will generate code to change the text colors.

  • Generated code example: textBox.setTextColors([[255,0,0][0,255,0]]);



Add Text Code Dialog (Background Color Tab)


The fourth tab contains a list that can be modified with the "+" and "-" buttons on the side. Clicking "+" will add a new color to the list and clicking "-" will remove the currently selected one. The color components are in the order red, green and blue. Clicking the color itself on the right will open a color wheel to freely select the color. If Set text background colors is checked this will generate code to change the text and button background colors.

  • Generated code example: textBox.setBackgroundColors([[255,255,255][255,255,255]]);
