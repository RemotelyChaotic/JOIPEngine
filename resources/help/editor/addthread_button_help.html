Add Thread Code

This button opens an overlay to add code snippets to the currently opened script in the Script Editor, to sleep the script thread.


For more information about the script API, please see the API Reference.



Add Thread Code Dialog


The sleep time for the generated code can be configured in the spinn box. If Skippable is checked it will generate code for a skippable sleep.

  • Generated code example: thread.sleep(120,true); // skippable 120s wait
