Add Thread Code

This button opens an overlay to add code snippets to the currently opened script in the Script Editor, to sleep the script thread and create or kill threads.


For more information about the script API, please see the API Reference.



Add Thread Code Dialog


The sleep time for the generated code can be configured in the spin box. If Skippable is checked it will generate code for a skippable sleep.

  • Generated code example: thread.sleep(120,true); // skippable 120s wait


Id for the thread to kill can be entered in the line edit below the sleep settings.

  • Generated code example: thread.kill("myThread");


Finally the id for the thread to create and the Resource of the thread can be entered in the line edits below the kill settings. The resource can also be browsed in the tree at the bottom.

  • Generated code example: thread.runAsynch("myResource", "MyId");
