Add Timer Code

This button opens an overlay to add code snippets to the currently opened script in the Script Editor, to manipulate the timer.


For more information about the script API, please see the API Reference.



Add Timer Code Dialog


Checking the Set time check box will generate code to set the timer to the amount configured in the spin box next to the check box.

  • Generated code example: timer.setTime(120);


Checking the Show timer or Hide timer check box will generate code to show or hide the timer respectively.

  • Generated code example for Show timer: timer.show();
  • Generated code example for Hide timer: timer.hide();


Checking or unchecking the Display time check box will generate code to show or hide the time display on the timer respectively.

  • Generated code example: timer.setTimeVisible(true);
  • Generated code example: timer.setTimeVisible(false);


Checking the Start or Stop check box will generate code to stop or start the timer respectively.

  • Generated code example for Start: timer.start();
  • Generated code example for Stop: timer.stop();


Checking the Wait for finish check box will generate code to wait on the timer, blocking and sleeping until the timer finishes.

  • Generated code example: timer.waitForTimer();
