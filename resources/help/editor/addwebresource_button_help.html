Add Online Resource

Open the web-overlay to add an Online Resource to the JOI-Project.



Web resource overlay


Only image and video type Online Resources can be handled, and the link must point to the image or video url directly. Links gated behind logins will not work. These Resources will not be saved on your disk and instead (down-)loaded into memory and displayed when needed.



This type of resource will disable this JOI-Project in offline mode and can cause player experience to drop if the internet connection is slow. This type of resource is not recommended to use.
