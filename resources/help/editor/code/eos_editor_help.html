EOS Editor

The script editor lets you modify an eos script Resources. It works similarly to the Milovana editor.


See the EOS Reference for help about eos scripting or check the help for the respective buttons.

  • audio.play: Audio Instruction
  • choice: Choice Instruction
  • disable: Disable Instruction
  • enable: Enable Instruction
  • end: End Instruction
  • eval: Eval Instruction
  • goto: Goto Instruction
  • if: If Instruction
  • image: Image Instruction
  • notification.create & notification.remove: Notification Create and Close Instructions
  • prompt: Prompt Instruction
  • say: Say Instruction
  • timer: Timer Instruction
