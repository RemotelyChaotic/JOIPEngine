Script Editor

The script editor lets you modify script Resources. It comes with a simple code highlighter and a debugging interface showing errors in the script file in a debugging session. The code editor is basically an extended notepad editor and allows the following actions:

  • Copy (Ctrl+C)
  • Paste (Ctrl+V)
  • Cut (Ctrl+X)
  • Undo (Ctrl+Z)
  • Redo (Ctrl+Y)
  • Select All (Ctrl+A)
  • Find (Ctrl+F)

Lines are marked with their index on the left side for easier navigation.


See the Script Reference for javascript help.
