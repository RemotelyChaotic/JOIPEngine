Script Editor

The script editor lets you modify script Resources. It comes with a simple code highlighter and a debugging interface showing errors in the script file in a debugging session. The code editor is basically an extended notepad editor and lets you copy (Ctrl+C), paste (Ctrl+V), cut (Ctrl+X), undo (Ctrl+Z), redo (Ctrl+Y) content. Lines are marked with their index on the left side for easier navigation.


See the Script Reference for javascript help.
