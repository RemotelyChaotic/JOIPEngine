Edit Dialogue Node Properties

This button opens an editor, to change a Dialogue's or Conditional Dialogue's properties.


Dialogue Property Editor Dialogue UI

The following properties can be chainged:

  • Id: The identifier of the Conditional Dialogue. Is enforced to be globally unique.
  • File: The Resource file the Dialog is stored in.
  • Skippable: Skippable flag. If set, the line of Dialogue is skippable.
  • Wait: The wait time in ms that determines how long the text is shown.
    Checking auto means the time is determined automatically based on the Text.
  • Text: The line of Dialogue shown in a TextBox.
  • Resource: A Sound type Resource. It is played when the Text is shown and stopped if the Text is skipped. If duration of the audio is longer than the Wait time, the text is shown for as long as the audio plays.
  • Condition (Conditional only): The condition under which the Dialogue is selected. This is UI eval code. The first condition that evaluates to true in a Dialogue will be used.
