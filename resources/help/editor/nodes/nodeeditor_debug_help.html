Node Debugging

The "Play" button starts the Nodes debugging session.


The current Node will be inserted into the list from the right and will be colored red in the Node view below. Already passed Nodes will be colored yellow.



Debugging View


Double click a Node to jump to the node in the Scene below.


The "Stop" button terminates the debugging session again.


Node previews in the debugger can't be modified except for splitters where you can select a possible path to steer the debugging flow towards. Scripts of the scenes are not taken into account with the debugging flow and jumps inside the scripts will not be considered.
