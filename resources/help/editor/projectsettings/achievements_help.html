Achievements

Shows and serves as an editor for Achievements of the JOI-Project.



Achievements Editor


Achievements have multiple properties that can be set. The properties depend on the achievement type:

  • Icon: The Resource serving as the achievement icon.
    Click to open the resource tree where you can select a Resource. Only Image and Video type Resources are accepted.
  • Name: The achievement name. Also serves as the ID.
  • Type: The Achievement type. Possible types are "Counting" and "Locked / Unlock". "Counting" achievements have a counter for the achievement progress while "Locked / Unlocked" achievements can only be unlocked or not.
  • Counter: The maximum number of steps needed before the achievement unlocks.
  • Describtion: The achievement describtion.
