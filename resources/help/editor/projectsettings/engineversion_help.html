Target Engine Version

Shows the JOIP-Engine version this JOI-Project was last saved in. If the engine version doesn't match the curent version, a warning will be shown. If the JOI-Project was saved in an earlier version, it should not have any impact to upgrade it, as the engine is backwards compatible. If you save a JOI-Project created in a later engine version, some features will be lost and might not be recoverable after saving.
