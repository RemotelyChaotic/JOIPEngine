Project Default Layout

Lets you set the default Layout used in the Player for the JOI-Project.


Custom .qml and .layout files will be listed here, if added as resources to the JOI-Project via the Resource Viewer.


If a JOI-Project contains no Layout file, the default Layout file will be used (based on engine version).


The button next to the selection can be used to create a new Layout file.


Some Scenes can overwrite this if configured in the Scene Node.
