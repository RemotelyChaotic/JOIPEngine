Plugin Subfolder

Lets you set the Plugin Subfolder for the JOI-Project.


Plugins are a list of JOIP-Projects that are loaded at run-time (not in the editor) additionally to this Project. Any Resources, Scenes and further plugins the Plugin defines can be used in this Project as well. If a name clashes, the last loaded Plugin's item that defines it is used effectively allowing a plugin to overwrite Resources, Scenes, etc. This override does not apply to direct properties of the Project itself (description, layout etc.). Achievements specifically are an exception and can be extended.


Plugins can bring their own Plugins which are loaded recursively.
