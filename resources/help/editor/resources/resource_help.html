Resource

Resources are the content files of a JOI-Project and must reside in the folder directory. Files outside the project directory cannot be used. Resources can represent files of different formats and types and are referenced in the application by their internal name / identifier, which does not need to equal the file name.


Resources can be renamed by double-clicking the name field in the Resource Tree. Renaming a resource changes it's internal identifier, but not the file name.


There's currently four differnt resource types:

  • Image: Images can be animated or static and are used for the Title Card and to display in the player via Media Player or as an Icon. Images can be online or offline. For online resources, the link must point to the image or video url directly. Links gated behind logins will not work. These Resources will not be downloaded and instead loaded and displayed when needed.
    Supported image formats are: bmp, cur, gif, icns, ico, jpeg, jpg, pbm, pgm, png, ppm, svg, svgz, tga, tif, tiff, wbmp, webp, xbm, xpm
  • Video: Videos can be displayed in the player via Media Player. Videos can be online or offline. Online videos only support a subset of the formats. For online resources, the link must point to the image or video url directly. Links gated behind logins will not work. These Resources will not be downloaded and instead loaded and displayed when needed.
    Supported video formats are: 3gp, asf, wmv, mpg, ts, au, avi, flv, mov, mp4 (online available), ogm, ogg, mkv, mka, nsc, nsv, nut, a52, dv, vid, ty, webm (online available)
  • Audio: Audio files can be played in the player via Media Player.
    Supported audio formats are: wav, dts, mp3, mp2, tta, tac, flac, dts, aac, xa
  • Other Formats: This resource type is used for various things in the project. The flow file is used as storage for the Node Editor's content, the js files are used for the Scene Scripts and the Project data is stored in json files. One thing they have in common is that all of them can be opened in any external text editor to edit the contents.
    Supported other formats are: json, xml (currently unused), js, flow



Online of resource will disable the JOI-Project in offline mode and can cause player experience to drop if the internet connection is slow. This type of resource is not recommended to use.
