Resource Tree

Shows Resources (images, videos, sounds, scripts etc.) you have added to the JOI-Project. The Resources are sorted by category and name, but the sorting can be changed by clicking on the tree header.


An alternative to adding Resources via the Add Resource Button, is to drag-and-drop the files from the explorer into the resource tree.


Resources can be renamed by double-clicking the name field. Renaming a resource changes it's internal identifier, but not the file name.
