Resource Viewer

Shows Resources (images, videos, sounds, scripts etc.) you have added to the JOI-Project. The Resources are sorted by category and name, but the sorting can be changed by clicking on the tree header.


The Resources can be displayed in two different views: The Tree View and the Explorer View.


The Resources are sorted by category and name, but in the Tree View, the sorting can be changed by clicking on the tree header.

Tree View

In the Tree View all Resources are displayed in a filesystem tree to provide an overview over all Resources. The columns show information such as the path in the JOI-Project's folder and Resource type. Hovering over a Resource shows you the Source String of the Resource.

Explorer View

In the Explorer View only the Resources in the currently viewed folder are displayed. Image and Video type Resources show a small preview of the content. Hovering over a Resource shows you the Source String of the Resource.

Adding Resources

An alternative to adding Resources via the Add Resource Button, is to drag-and-drop the files from the operation system's explorer into the Resource Viewer.

Editing Resources

Resources can be renamed by double-clicking the name field. Renaming a resource changes it's internal identifier, but not the file name.
