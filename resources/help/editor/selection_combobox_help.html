View Selection

The editor has two planes to display views and you can swap what they display with this combo-box. The planes can show the following views:


  • Resource Manager: Shows Resources (images, videos, sounds, scripts etc.) in the Resource Tree you have added to the JOI-Project.
  • Resource View: Shows the currently selected Resource from the Resource Tree. Resources can be replayed or simply viewed in this preview.
  • Scene Node Editor: Allows to edit the JOI-Project's flow in the Node Editor.
  • Code Editor: This Editor lets you edit javascript Resources and debug existing scenes.
