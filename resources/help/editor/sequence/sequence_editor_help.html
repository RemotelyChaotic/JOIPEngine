Sequence Editor

The sequence editor allows for editing a sequence and its layers.


At the top of the editor, the controls allow to change certain editor properties. These are, from left to right:

  • Zoom Level: Sets the zoom level. You can either select a pre-defined value, or enter a custom value.
  • Grid: Sets the interval that the cursor will be constrained to in milliseconds. It allows for more controlled placement of Instructions. Setting a value of 0 or 1 will disable the grid and enable free placement of instructions.
  • Cursor Time: Shows the current time stamp of the cursor.



Error Controls


Below the controls, the list of layers, present in the sequence, can be edited. The header of the layer is to the left. Grabbing the empty space below the properties allows for re-ordering the layers. The properties are, from top to bottom:

  • Layer Type: The type of instructions that can be placed on the layer. Each layer is limited to one type of instructions. The types are: Metronome Commands, Toy Commands, Resource Commands, Text Commands and Script Commands.
  • Reference Object: This defines the object in the Layout that the commands will be sent to. This can be thought of as the object in Scene Scripts whose methods are called. Multiple layers can reference the same object.



Sequence Layer


The timeline in the right part of the editor can be used to place Instructions. Selecting an instruction opens the editor for the specific instruction.
