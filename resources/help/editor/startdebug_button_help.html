Start Debugging

Clicking the Start Debugging button will start a debugging session with the currently opened Scene script in the Script Editor.



The JOI-Project needs to be saved for debugging.

This allows to test Scripts for functionality and to resolve exceptions. Exceptions that occur will be displayed next to the line number on the line that the exception occured. The exception reason will be displayed in red next to the code as well as text in the player. Keep in mind that the PlayerStorage will most likely not be populated, if a different script does it.
