Project Title Card

Title card of a JOI-Project click to open it in the editor or the player.


There's a set of icons that show certain project properties that could affect gameplay experience:

  • : Project contains sound or video and might be unexpectedly loud. If viewing in public without headphones, you might want to mute the engine.
  • : Project might require an active internet connection, because it contains web-based resources. JOI-Project is not viewable in offline mode.


Opening the detailed view, lets you view the Project describtion and the tagged Fetishes.
