Debug Overlay

Toggle the debug overlay. While active, the overlay collects all messages that are normally sent to the debug output and displays them. Also contains a command line interface to control and debug the engine.



Multiple instances of the application share the same configuration file.
If you use different versions of the application, your settings for all instances will change.


The log is never cleareed while the application is running. If you leave the engine running for an extremely long time with this active, memory will eventually run out and the engine will crash.
