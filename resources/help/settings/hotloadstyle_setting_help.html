Hotload Style

Configure the stlye to be hotloaded if the style file changes on the system during application runtime. This will only cause the application style to be hotloaded, as the qml-style can not be hotloaded at runtime. Useful for testing styles during custom style development.



Multiple instances of the application share the same configuration file.
If you use different versions of the application, your settings for all instances will change.


Hotloading a style may cause significant lag while the new style is being applied.
