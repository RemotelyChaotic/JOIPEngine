Metronome Default Commands

Sets the default commands that are sent to a connected device by the metronome, if the project does not overwrite the commands in the Project Settings > Metronome Toy Commands. Multiple of these can be sent simultaniously, if the connected device does not suport a command, it will be ignored.


The possible options are:

  • Vibrate: Sends ScalarCmd to the device alternating between 25% (mid-tick) and 75% (on tick) strength.
  • Linear: Sends LinearCmd to the device alternating between depth 0% and 100% on every other tick.
  • Rotate: Sends RotateCmd to the device alternating between clockwise and counter-clockwise rotation on every other tick.



Multiple instances of the application share the same configuration file.
If you use different versions of the application, your settings for all instances will change.
