Window Mode

Toggles window mode for the application.


Available values are:

  • Fullscreen: Application fills a screen.
  • Borderless Window: Application respects the selected Resolution but is displayed without a header.
  • Windowed: Application is shown as moveable, resizable window and respects the selected Resolution.


Multiple instances of the application share the same configuration file.
If you use different versions of the application, your settings for all instances will change.
