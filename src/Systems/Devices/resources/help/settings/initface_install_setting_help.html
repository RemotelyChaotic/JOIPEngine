Initface Install Location

Select the install location of Initface Central (https://intiface.com/central/).
The application will try to start and connect to Initface Central via the Port Setting or connect to an already running Initface Central instance.
If unsuccessful, the application will spawn an Initface Engine (https://github.com/intiface/intiface-engine) instance that is shipped with the application as a plugin and connect it to the configured Port.
Thus if you wish to use a custom Initface Central installation, you need to start it before starting the JOIP Engine and launch the server (click the big play button) or set this path to the correct intiface_central.exe install.



Multiple instances of the application share the same configuration file.
If you use different versions of the application, your settings for all instances will change.
